# dotfiles

Arch Linux · AwesomeWM · Catppuccin Mocha

## Setup

```bash
git clone https://github.com/WildRelation/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## Stack

| Tool | Purpose |
|------|---------|
| AwesomeWM | Window manager |
| Kitty | Terminal |
| Zsh + starship | Shell + prompt |
| Rofi | App launcher |
| Picom | Compositor |
| Conky | System monitor |
| Fastfetch | System info |
| Catppuccin Mocha | Theme (everywhere) |
| JetBrains Mono | Font |
| Papirus Dark | Icons |
